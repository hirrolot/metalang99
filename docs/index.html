The Epilepsy Standard Library¶

The Epilepsy Standard library exports a set of macros implemented using the Epilepsy metalanguage.

eval.h¶

Defines

EPILEPSY_EVAL(...)¶

Evaluates a metaprogram.

lang.h¶

Defines

EPILEPSY_CALL(...)¶

Invokes a metafunction with arguments.

EPILEPSY_APPLY(f, ...)¶

Applies arguments to f.

EPILEPSY_APPLY_2(f, a, b)¶

Applies a and b to f.

EPILEPSY_APPLY_WITH_ARITY(f, arity, ...)¶

Applies arguments to f with the arity arity.

v(...)¶

Represents a normal form of an Epilepsy term.

EPILEPSY_DESUGAR(f, ...)¶

Desugars calls to f.

control.h¶

Control expressions.

Defines

EPILEPSY_IF(cond, x, y)¶

If cond is true, expands to x, otherwise y.

EPILEPSY_IF_LAZY(cond, f, g, ...)¶

Lazy #EPILSPY_IF.

record.h¶

Record types.

Defines

EPILEPSY_RECORD(...)¶

Constructs an instance of a record type.

EPILEPSY_GET(record, i)¶

Extracts a field of a record type.

choice.h¶

Choice types.

Defines

EPILEPSY_CHOICE(tag, ...)¶

Constructs an instance of a choice type.

EPILEPSY_CHOICE_EMPTY(tag)¶

The same as EPILEPSY_CHOICE but does not require additional arguments.

EPILEPSY_MATCH(choice, matcher)¶

Matches the instance choice of a choice type.

EPILEPSY_MATCH_WITH_ARGS(choice, matcher, ...)¶

The same as EPILEPSY_MATCH but supplies additional arguments to all branches.

Logical negation.

Logical conjunction.

Logical inclusive OR.

Logical exclusive OR.

EPILEPSY_CONST_TRUE(...)¶

either.h¶

Defines

The left value x.

EPILEPSY_EitherRight(x)¶

The right value x.

EPILEPSY_EitherIsLeft(either)¶

1 if either contains a left value, otherwise 0.

EPILEPSY_EitherIsRight(either)¶

1 if either contains a right value, otherwise 0.

maybe.h¶

An optional value.

Defines

EPILEPSY_MaybeJust(x)¶

Some value x.

EPILEPSY_MaybeNothing()¶

No value.

EPILEPSY_MaybeIsJust(maybe)¶

1 if maybe contains some value, otherwise 0.

EPILEPSY_MaybeIsNothing(maybe)¶

1 if maybe contains no value, otherwise 0.

list.h¶

Lists.

Defines

EPILEPSY_Cons(x, xs)¶

Prepends x to xs.

EPILEPSY_Nil()¶

The empty list.

EPILEPSY_List(...)¶

Constructs a list from its arguments.

EPILEPSY_ListUnwrap(list)¶

Places all the items in list as-is.

EPILEPSY_ListIsEmpty(list)¶

Checks list for emptiness.

EPILEPSY_ListGet(list, i)¶

Gets an i -indexed element.

EPILEPSY_ListFoldr(list, f, init)¶

Performs a right-associative fold over list.

EPILEPSY_ListFoldl(list, f, init)¶

Performs a left-associative fold over list.

EPILEPSY_ListFoldl1(list, f)¶

The same as EPILEPSY_ListFoldl but treats the first element of list as the initial value.

EPILEPSY_ListIntersperse(list, item)¶

Intersperses item between the items in list.

EPILEPSY_ListPrependToAll(list, item)¶

Prepends item to all items in list.

EPILEPSY_ListMap(list, f)¶

Maps all the elements in list with f.

EPILEPSY_ListEq(list, other, compare)¶

Checks list and other for equality.

EPILEPSY_ListApply(list, f)¶

Applies all the items in list to f.
